📚 বহুমুখী তথ্যকোষ

নোট: Nikosh/NikoshBan/Siyam Rupali ফন্ট আপনার ডিভাইসে ইনস্টল থাকলে সেটিই ব্যবহার হবে। অথবা "🔤 ফন্ট আপলোড করুন" বাটনে চেপে ফন্ট ফাইল (.ttf/.otf) সরাসরি অ্যাপে যুক্ত করে দিন — একবার আপলোড করলে পরবর্তীতে প্রতিবার এই ফোনে/এই অ্যাপে অটোমেটিক লোড হবে।

🔤 কাস্টম ফন্ট আপলোড ব্যবস্থা

Android ফোনে সিস্টেম-ওয়াইড ফন্ট ইনস্টল না করেই, নিচে আপনার ডাউনলোড করা .ttf/.otf ফাইল সিলেক্ট করুন — অ্যাপ নিজেই ফন্টটি সংরক্ষণ করে সাথে সাথে ব্যবহার শুরু করবে। ফন্ট ফাইল কীভাবে জোগাড় করবেন তার ধাপ নিচে দেওয়া আছে।

📥 Android ফোনে ফন্ট ফাইল কীভাবে জোগাড় করবেন? (ধাপে ধাপে)
  1. ফোনের Chrome ব্রাউজারে omicronlab.com-এ যান (Avro Keyboard-এর অফিসিয়াল সাইট, যেখান থেকে Bangla ফন্টগুলো বিনামূল্যে বিতরণ করা হয়) এবং "Fonts" বা "Downloads" সেকশনে যান।
  2. SiyamRupali.ttf / Nikosh.ttf / NikoshBan.ttf ফাইল খুঁজে ডাউনলোড করুন — এগুলো সাধারণত .zip আকারে থাকে।
  3. ডাউনলোড হওয়া .zip ফাইলটি ফোনের "Files" বা "My Files" অ্যাপ দিয়ে খুলে Extract করুন (প্রয়োজনে "ZArchiver" জাতীয় ফ্রি অ্যাপ ব্যবহার করুন)।
  4. এক্সট্র্যাক্ট হওয়ার পর .ttf ফাইলটি সাধারণত Downloads ফোল্ডারেই থাকবে।
  5. এই অ্যাপে ফিরে এসে উপরের নির্দিষ্ট বক্সে (Siyam Rupali / Nikosh / NikoshBan) "Choose File" চেপে Downloads ফোল্ডার থেকে সেই .ttf ফাইলটি সিলেক্ট করুন।
  6. আপলোড সম্পন্ন হলে স্ট্যাটাসে "✅ সক্রিয়" দেখাবে এবং উপরের ফন্ট ড্রপডাউন থেকে সেই ফন্ট সিলেক্ট করলেই পুরো অ্যাপে সেটি প্রয়োগ হবে। এটি ফোনে/ব্রাউজারেই সংরক্ষিত থাকে, তাই একবার আপলোড করলে পরের বার অ্যাপ খুললেও আর লাগবে না।

বিকল্প: "Avro Keyboard" বা "Bangla Font Pack" লিখে Google-এ সার্চ করলেও অফিসিয়াল ফ্রি সোর্স থেকে ডাউনলোড লিংক পাবেন। কপিরাইট সুরক্ষিত থার্ড-পার্টি সোর্স এড়িয়ে সবসময় অফিসিয়াল/ফ্রি-লাইসেন্সড সোর্স থেকে ডাউনলোড করুন।

সংখ্যাRoman Numeral

১০০-এর বেশি বড় রোমান সংখ্যা লেখার নিয়ম

  • মূল প্রতীক: I=１, V=৫, X=১০, L=৫০, C=১০০, D=৫০০, M=১০০০।
  • একই প্রতীক পরপর সর্বোচ্চ ৩ বার লেখা যায় (যেমন III=৩), তবে V, L, D কখনো পুনরাবৃত্তি হয় না।
  • ছোট প্রতীক বড় প্রতীকের আগে বসলে বিয়োগ হয় (যেমন IV=৪, IX=৯, XL=৪০, XC=৯০, CD=৪০০, CM=৯০০)।
  • ১০০০-এর বেশি সংখ্যার জন্য প্রতীকের উপর একটি রেখা (vinculum) দিলে তার মান ১০০০ গুণ হয়ে যায় — যেমন V̄ = ৫,০০০, X̄ = ১০,০০০, M̄ = ১০,০০,০০০।
  • উদাহরণ: ২০২৬ = MMXXVI, ১৯৯৯ = MCMXCIX, ৪৯৯৯ = MMMMCMXCIX বা IV̄CMXCIX।
  • রোমান পদ্ধতিতে শূন্য (0) বা ঋণাত্মক সংখ্যা প্রকাশের কোনো প্রতীক নেই।
